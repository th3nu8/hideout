# Hideout

An open-source unblocked platform with 1000+ games, proxy browser, global chat, ai chatbot, and more.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:8080`

## Deploy

### Install
```bash
npm install
```

### Build
```bash
npm run build
```

## .env File
```bash
VITE_SUPABASE_PROJECT_ID=""
VITE_SUPABASE_PUBLISHABLE_KEY=""
VITE_SUPABASE_URL=""
VITE_GROQ_API_KEY=""
```
